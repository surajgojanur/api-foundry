# API Foundry 2-Minute Demo Script

## 1. Opening line

"Competitor websites have valuable business data, but they do not expose APIs. API Foundry turns those websites into live competitor intelligence APIs."

## 2. Problem

"Today, teams manually check competitor prices, offers, stock, and announcements. That’s slow, inconsistent, and hard to operationalize."

## 3. Solution

"We use Anakin to discover and extract competitor web data, generate reusable APIs, detect changes between snapshots, and produce deterministic AI-style business insights."

## 4. Demo steps

1. Open dashboard (`/`) and show Market Intelligence panel.
2. Open Zepto project (`/projects/zepto`).
3. Show generated feed endpoint and API panel.
4. Show changes endpoint and timeline with severity.
5. Trigger refresh endpoint.
6. Show updated change summary and insight cards.
7. Call insights endpoint (`/api/v1/projects/zepto/insights`).
8. Open compare page (`/compare`) and show:
- most aggressive competitor
- most stable competitor
- recommended monitoring priority.

## 5. Closing line

"Everyone scrapes pages. API Foundry turns competitor websites into API products and change intelligence."
